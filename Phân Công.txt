 Hiếu:
 - Đăng nhập, đăng xuất, đổi mật khẩu
 - Khách hàng:
	+ Tra cứu sản phẩm, Loại sản phẩm, 
	+ Đặt hàng, hủy hàng
 - Quản lý danh mục:
	- Loại hàng, => DONE DB, BE [CRUD]
	- Mặt hàng, => DONE DB, BE [CRUD]
	- Khách hàng, => DONE DB
	- Nhà Sản xuất, => DONE DB, BE [CRUD]
	- Loai khách hàng,  => DONE DB, BE [CRUD]
	- Nhân viên,  => DONE DB
	- phòng ban => DONE DB, BE [CRUD]
 - Thiết kế, cài đặt cơ sở dữ liệu.

quản lý vị trí lưu trữ sản phẩm trên ngăn kệ
 - Khảo sát, phân tích, thiết kế hệ thống.
